👋 首次使用前请配置 API Key，配置一次后无需重复填写。欧美/日韩市场用 Claude Key，中国市场用豆包 Key，填一个即可运行。
🇨🇳 国内用户注意：Claude 官方地址在国内无法直连（会报 Failed to fetch）。需要填写「Claude 接口地址」（中转站 URL），或者直接只用豆包 Key 跳过 Claude。
官方：console.anthropic.com → API Keys · 中转站填入中转提供的 key 即可
填中转站根域名即可，如 https://api.xxx.com · 留空则直连官方（国内会报错）
